- 数据库：MySQL 8.0+

1. 依次执行 SQL 脚本：

## 3. 关键测试用例

### 用例1：创建订单自动锁车（规则A）
- 前置：选取状态=在库车辆VIN。
- 操作：调用 `sp_create_sales_order` 创建订单。
- 预期：订单创建成功，车辆状态变为“已锁定”。

### 用例2：订单完成自动售出（规则B）
- 前置：存在状态“待交付”的订单。
- 操作：`UPDATE sales_orders SET order_status='已完成' ...`。
- 预期：对应车辆状态变为“已售出”，并写入 `sold_at` 与 `delivered_at`。

### 用例3：库存预警（Q7）
- 操作：执行 Q7 查询。
- 预期：返回在库数量小于安全库存阈值的车型清单。

### 用例4：顾问业绩榜（Q2）
- 操作：执行 Q2 查询，按月/季度查看。
- 预期：输出顾问订单数、销售额、毛利及排名。

### 用例5：Python 前台下单
- 操作：销售前台 -> 创建销售订单。
- 预期：存储过程被调用，订单与明细写入成功。
