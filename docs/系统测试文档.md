## 1. 测试目标与范围

### 1.1 测试目标

1. 验证核心业务规则 A/B/C 在数据库层可被强制执行。  
2. 验证 Q1~Q7（及扩展 Q8）查询满足业务统计需求。  
3. 验证 Python 控制台应用三大模块（销售前台/库存管理/报表中心）可联通数据库并正确执行业务流程。  
4. 形成“步骤-结果-截图-分析”可追溯证据，满足课程作业提交要求。

### 1.2 测试范围

- SQL 脚本：`sql/01_create_schema.sql` ~ `sql/07_queries.sql`
- 应用程序：`python_app/main.py`
- 关键对象：视图、触发器、存储过程、复杂查询、菜单功能

## 2. 测试环境

## 4. 关键测试用例（含截图点与结果分析）

### TC-01 创建订单自动锁车（规则A）

- **测试目标**：验证下单时仅允许“在库”车辆，并自动锁车。  
- **前置条件**：存在 `status='在库'` 的 VIN。  
- **执行步骤**：调用 `sp_create_sales_order(...)` 或 Python“创建销售订单”。  
- **预期结果**：订单创建成功，车辆状态由“在库”变为“已锁定”。  
- **截图点**：下单成功回显 + `inventory_vehicles` 状态查询结果。  
- **结果分析**：证明触发器 `trg_lock_car_on_order` 生效，满足规则A。  

### TC-02 订单完成自动售出（规则B）

- **测试目标**：验证订单完成时库存状态联动。  
- **前置条件**：存在“待交付”订单。  
- **执行步骤**：`UPDATE sales_orders SET order_status='已完成' WHERE order_no='...'`。  
- **预期结果**：对应车辆状态变“已售出”，并写入 `sold_at`、`delivered_at`。  
- **截图点**：订单更新结果 + 车辆状态与时间戳查询结果。  
- **结果分析**：证明触发器 `trg_update_inventory_on_delivery` 生效，满足规则B。  

### TC-03 订单取消库存回退（扩展一致性）

- **测试目标**：验证取消订单时库存回退逻辑。  
- **前置条件**：存在“待定金/待交付”且车辆状态“已锁定”的订单。  
- **执行步骤**：将订单状态更新为“已取消”。  
- **预期结果**：车辆状态由“已锁定”回退为“在库”。  
- **截图点**：订单状态变更 + 车辆状态回退结果。  
- **结果分析**：说明触发器处理了异常业务流转，避免库存脏状态。  

### TC-04 非法下单回滚（事务一致性）

- **测试目标**：验证过程失败时回滚，不产生脏数据。  
- **前置条件**：构造非法输入（VIN不存在/非在库/总金额<0）。  
- **执行步骤**：调用 `sp_create_sales_order`。  
- **预期结果**：抛出错误；`sales_orders` 与 `sales_order_items` 无新增脏记录。  
- **截图点**：报错信息 + 下单前后记录数对比。  
- **结果分析**：证明存储过程事务与异常处理有效。  

### TC-05 Q2 顾问业绩排行

- **测试目标**：验证按月/季度业绩统计与排名准确。  
- **执行步骤**：执行 `07_queries.sql` 的 Q2。  
- **预期结果**：输出顾问订单数、销售额、毛利及排名。  
- **截图点**：Q2 查询结果。  
- **结果分析**：满足任务书“报表中心-业绩榜”与 Q2 需求。  

### TC-06 Q7 库存预警

- **测试目标**：验证安全库存阈值预警能力。  
- **执行步骤**：执行 `07_queries.sql` 的 Q7 或 Python“查看库存预警报表”。  
- **预期结果**：返回 `in_stock_count < safety_stock` 的车型清单。  
- **截图点**：Q7 SQL结果或应用端预警结果。  
- **结果分析**：满足规则C与库存管理功能要求。  

### TC-07 Python 前台下单全链路

- **测试目标**：验证“应用层操作→过程调用→数据库落库”链路。  
- **执行步骤**：`main.py` 进入销售前台并创建销售订单。  
- **预期结果**：控制台显示新订单号；订单与明细均写入成功。  
- **截图点**：菜单操作过程 + 成功回显 + 数据库核验结果。  
- **结果分析**：证明应用层与数据库高级对象集成正确。
